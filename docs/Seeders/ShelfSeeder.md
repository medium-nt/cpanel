# ShelfSeeder

**Файл:** `database/seeders/ShelfSeeder.php`

## Назначение сида

Создание базового набора полок для организации складского хранения.

## Создаваемые данные

- **Полки:** 3 записи
    1. первая
    2. вторая
    3. третья

## Особенности генерации

- Использует метод `firstOrCreate` для предотвращения дублирования
- Создает простую нумерацию полок для удобства использования

## Зависимости от других сидов

Не имеет зависимостей.

## Выполняемые операции

Создает 3 полки, если они еще не существуют в системе.

## Тестовые данные

```
Полки на складе:
1. первая
2. вторая
3. третья
```

## Примечания

Полки используются для организации хранения материалов и готовой продукции на
складе.
