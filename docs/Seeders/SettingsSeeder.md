# SettingsSeeder

**Файл:** `database/seeders/SettingsSeeder.php`

## Назначение сида

Создание начальных настроек системы для корректной работы приложения.

## Создаваемые данные

- **Настройки:** 14 записей системных настроек

## Особенности генерации

- Использует метод `firstOrCreate` для предотвращения перезаписи существующих
  настроек
- Создает критически важные настройки для работы системы
- Включает настройки интеграции с маркетплейсами

## Зависимости от других сидов

Не имеет зависимостей.

## Выполняемые операции

Создает следующие настройки:

1. **Время работы:**
    - начало рабочего дня: 07:00
    - конец рабочего дня: 20:00

2. **Функциональность:**
    - включен ли график работы: 0 (выключен)
    - включены ли смены: 0 (выключено)

3. **Интеграция с маркетплейсами:**
    - API ключ Wildberries: пустой
    - API ключ Ozon: пустой
    - Seller ID Ozon: пустой

4. **Ограничения:**
    - макс. заказов для швеи: 7
    - макс. заказов для закройщика: 7
    - дневной лимит для закройщика: 1

5. **Приоритеты:**
    - приоритет заказов: by_date (по дате)

6. **Штрафы:**
    - за опоздание на смену: 0
    - за незакрытую смену: 0
    - за отмену заказа: 0

## Тестовые данные

Все настройки имеют значения по умолчанию, обеспечивающие базовую
функциональность системы.

## Примечания

Настройки API ключей intentionally пустые - должны быть заполнены
администратором после развертывания системы.
