# MarketplaceItemSeeder

**Файл:** `database/seeders/MarketplaceItemSeeder.php`

## Назначение сида

Наполнение базы данных товарами для маркетплейсов (Wildberries, Ozon) с
различными характеристиками.

## Создаваемые данные

- **Товары:** 224 товара (2 типа тканей × 7 ширин × 16 высот)
- **Расход материалов:** 224 записи о расходе материалов
- **SKU:** 448 записей SKU (по 2 на каждый товар для разных маркетплейсов)

## Особенности генерации

- Использует комбинации всех возможных вариантов размеров для создания полного
  каталога
- Автоматически связывает каждый товар с соответствующим материалом (Бамбук →
  material_id=1, Лен → material_id=2)
- Создает пустые SKU для каждого товара на двух маркетплейсах

## Зависимости от других сидов

- `MaterialSeeder` - требует наличия материалов с ID 1 (Бамбук) и ID 2 (Лен)

## Выполняемые операции

1. Создает товары со всеми комбинациями:
    - Типы тканей: Бамбук, Лен
    - Ширины: 200, 300, 400, 500, 600, 700, 800 мм
    - Высоты: от 220 до 295 мм с шагом 5 мм
2. Для каждого товара создает запись о расходе материалов (количество = 1)
3. Для каждого товара создает SKU для двух маркетплейсов (ID 1 и 2)

## Тестовые данные

```
Типы тканей: Бамбук, Лен
Ширины: [200, 300, 400, 500, 600, 700, 800]
Высоты: [220, 225, 230, 235, 240, 245, 250, 255, 260, 265, 270, 275, 280, 285, 290, 295]
Маркетплейсы: ID 1, ID 2
```
