# MovementMaterialSeeder

**Файл:** `database/seeders/MovementMaterialSeeder.php`

## Назначение сида

Создание записей о движении материалов (приход/расход) в рамках заказов.

## Создаваемые данные

- **Движения материалов:** 4 записи
    - 3 записи по заказу №1 (приход материалов)
    - 1 запись по заказу №2 (заказанное количество)

## Особенности генерации

- Демонстрирует движение материалов между заказами
- Показывает как фактическое количество, так и заказанное
- Использует реальные материалы (ткани и упаковку)

## Зависимости от других сидов

- `MaterialSeeder` - требует наличия материалов с ID 1, 2, 3
- `OrderSeeder` - требует наличия заказов с ID 1, 2

## Выполняемые операции

Создает движения материалов:

1. Заказ 1: 100 погонных метров Бамбука
2. Заказ 1: 50 погонных метров Льна
3. Заказ 1: 10 коробок упаковки
4. Заказ 2: 0 фактических, 20 заказанных погонных метров Бамбука

## Тестовые данные

```
Движение 1:
- material_id: 1 (Бамбук)
- quantity: 100
- order_id: 1

Движение 2:
- material_id: 2 (Лен)
- quantity: 50
- order_id: 1

Движение 3:
- material_id: 3 (Коробка)
- quantity: 10
- order_id: 1

Движение 4:
- material_id: 1 (Бамбук)
- quantity: 0
- ordered_quantity: 20
- order_id: 2
```
