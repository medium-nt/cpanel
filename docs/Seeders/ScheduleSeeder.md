# ScheduleSeeder

**Файл:** `database/seeders/ScheduleSeeder.php`

## Назначение сида

Создание тестового графика работы для демонстрации системы планирования смен.

## Создаваемые данные

- **Записи в графике:** 5 записей для пользователя с ID 3
    - 2 прошедшие смены
    - 3 будущие смены

## Особенности генерации

- Все записи создаются для одного пользователя (ID 3 - тестовая швея)
- Используются относительные даты (сейчас +/- дни)
- Создает mixins как прошедшие, так и будущие смены

## Зависимости от других сидов

- `UserSeeder` - требует наличия пользователя с ID 3

## Выполняемые операции

Создает 5 записей в графике работы:

1. вчера
2. через 2 дня
3. позавчера
4. через 3 дня
5. через 4 дня

## Тестовые данные

```
График для пользователя ID 3:
1. Дата: вчера (-1 день)
2. Дата: через 2 дня (+2 дня)
3. Дата: позавчера (-2 дня)
4. Дата: через 3 дня (+3 дня)
5. Дата: через 4 дня (+4 дня)
```

## Примечания

График используется для системы планирования и учета рабочего времени
сотрудников.
