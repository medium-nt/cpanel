# SupplierSeeder

**Файл:** `database/seeders/SupplierSeeder.php`

## Назначение сида

Создание тестовых поставщиков материалов для системы.

## Создаваемые данные

- **Поставщики:** 10 записей

## Особенности генерации

- Использует фабрику моделей для генерации реалистичных данных
- Автоматически создает разнообразных поставщиков с тестовыми данными

## Зависимости от других сидов

Требует наличия Factory для модели Supplier.

## Выполняемые операции

Создает 10 поставщиков с использованием фабрики моделей.

## Тестовые данные

Данные генерируются автоматически через фабрику и включают:

- Название компании
- Контактную информацию
- Адрес
- Прочие реквизиты

## Примечания

Использование фабрики позволяет получать разнообразные и реалистичные тестовые
данные для каждого запуска сида.
