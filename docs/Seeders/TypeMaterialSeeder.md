# TypeMaterialSeeder

**Файл:** `database/seeders/TypeMaterialSeeder.php`

## Назначение сида

Создание базовой классификации типов материалов в системе.

## Создаваемые данные

- **Типы материалов:** 3 записи
    1. Тюль
    2. Аксессуары
    3. Упаковка

## Особенности генерации

- Создает основные категории материалов
- Простая структура - только название типа

## Зависимости от других сидов

Не имеет зависимостей, должен выполняться до `MaterialSeeder`.

## Выполняемые операции

Создает 3 типа материалов для классификации.

## Тестовые данные

```
Типы материалов:
1. Тюль - для ткани
2. Аксессуары - для фурнитуры и декоративных элементов
3. Упаковка - для коробок и упаковочных материалов
```

## Примечания

Типы материалов используются для организации каталога материалов и учета запасов
по категориям.
