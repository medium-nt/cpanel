# RoleSeeder

**Файл:** `database/seeders/RoleSeeder.php`

## Назначение сида

Создание базовых ролей пользователей в системе.

## Создаваемые данные

- **Роли:** 5 записей
    1. seamstress - Швея
    2. storekeeper - Кладовщик
    3. admin - Администратор
    4. cutter - Закройщик
    5. otk - Сотрудник ОТК (Отдел технического контроля)

## Особенности генерации

- Использует метод `firstOrCreate` для предотвращения дублирования
- Создает минимально необходимый набор ролей для работы системы

## Зависимости от других сидов

Не имеет зависимостей, должен выполняться первым.

## Выполняемые операции

Создает 5 ролей, если они еще не существуют в системе.

## Тестовые данные

```
Роли в системе:
1. seamstress - Швея
2. storekeeper - Кладовщик
3. admin - Администратор
4. cutter - Закройщик
5. otk - Сотрудник ОТК
```

## Примечания

Роли являются основой для системы доступа и прав пользователей в приложении.
Каждая роль определяет доступные пользователю функции и интерфейсы.
