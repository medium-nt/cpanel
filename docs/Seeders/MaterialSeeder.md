# MaterialSeeder

**Файл:** `database/seeders/MaterialSeeder.php`

## Назначение сида

Создание базового набора материалов для производства штор.

## Создаваемые данные

- **Материалы:** 3 записи
    - 2 типа ткани (Бамбук, Лен)
    - 1 тип упаковки (Коробка)

## Особенности генерации

- Создает основные материалы, используемые в производстве
- Указывает реальные характеристики материалов (высота, единицы измерения)
- Привязывает материалы к соответствующим типам

## Зависимости от других сидов

- `TypeMaterialSeeder` - требует наличия типов материалов с ID 1 (Тюль) и ID 3 (
  Упаковка)

## Выполняемые операции

Создает 3 материала:

1. Бамбук (тюль, высота 200, измеряется в погонных метрах)
2. Лен (тюль, высота 225, измеряется в погонных метрах)
3. Коробка 200х20х20 (упаковка, измеряется в штуках)

## Тестовые данные

```
Материал 1:
- title: Бамбук
- type_id: 1 (Тюль)
- height: 200
- unit: п.м. (погонный метр)

Материал 2:
- title: Лен
- type_id: 1 (Тюль)
- height: 225
- unit: п.м. (погонный метр)

Материал 3:
- title: Коробка 200х20х20
- type_id: 3 (Упаковка)
- height: 0
- unit: шт. (штуки)
```
