# MarketplaceOrderSeeder

**Файл:** `database/seeders/MarketplaceOrderSeeder.php`

## Назначение сида

Создание тестовых заказов с маркетплейсов для демонстрации системы обработки
заказов.

## Создаваемые данные

- **Заказы:** 2 записи
    - 1 заказ с маркетплейса ID 1 (FBO)
    - 1 заказ с маркетплейса ID 2 (FBS)

## Особенности генерации

- Создает заказы с разными типами выполнения (FBO/FBS)
- Использует разные статусы для демонстрации жизненного цикла заказа
- Генерирует реалистичные номера заказов

## Зависимости от других сидов

Не имеет прямых зависимостей, но ожидает наличие маркетплейсов в системе.

## Выполняемые операции

Создает 2 заказа:

1. Заказ A-12345 (FBO, новый статус)
2. Заказ WB-12-ZZ-123 (FBS, в обработке)

## Тестовые данные

```
Заказ 1:
- order_id: A-12345
- marketplace_id: 1
- fulfillment_type: FBO
- status: 0 (новый)

Заказ 2:
- order_id: WB-12-ZZ-123
- marketplace_id: 2
- fulfillment_type: FBS
- status: 1 (в обработке)
```

## Примечания

- FBO (Fulfillment by Operator) - выполнение силами маркетплейса
- FBS (Fulfillment by Seller) - выполнение силами продавца
