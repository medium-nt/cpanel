# MarketplaceOrderItemSeeder

**Файл:** `database/seeders/MarketplaceOrderItemSeeder.php`

## Назначение сида

Создание тестовых позиций в заказах с маркетплейсов для демонстрации работы
системы заказов.

## Создаваемые данные

- **Позиции заказов:** 3 записи
- Заказ №1: 2 товара
- Заказ №2: 1 товар (3 штуки)

## Особенности генерации

- Жестко заданные значения для воспроизводимости тестовых сценариев
- Использует конкретные ID товаров и заказов
- Все позиции имеют статус 0 (новые)

## Зависимости от других сидов

- `MarketplaceOrderSeeder` - требует наличия заказов с ID 1 и 2
- `MarketplaceItemSeeder` - требует наличия товаров с ID 1, 2, 3

## Выполняемые операции

Создает 3 позиции товаров в заказах:

1. Заказ 1, товар 1, количество 1, цена 10.99
2. Заказ 1, товар 2, количество 1, цена 12.00
3. Заказ 2, товар 3, количество 3, цена 10.99

## Тестовые данные

```
Позиция 1:
- order_id: 1
- item_id: 1
- quantity: 1
- price: 10.99

Позиция 2:
- order_id: 1
- item_id: 2
- quantity: 1
- price: 12.00

Позиция 3:
- order_id: 2
- item_id: 3
- quantity: 3
- price: 10.99
```
