# Таблицы склада

## suppliers - Поставщики

**Файл миграции:** `2025_04_03_050040_create_suppliers_table.php`

### Структура таблицы:

| Колонка    | Тип         | Описание                    | Индекс/Ключ |
|------------|-------------|-----------------------------|-------------|
| id         | bigint      | ID поставщика               | PRIMARY     |
| title      | string(255) | Название поставщика         | UNIQUE      |
| phone      | string(255) | Телефон                     | UNIQUE      |
| address    | string(255) | Адрес                       | UNIQUE      |
| comment    | text        | Комментарий                 | NULLABLE    |
| created_at | timestamp   | Дата создания               |             |
| updated_at | timestamp   | Дата обновления             |             |
| deleted_at | timestamp   | Дата удаления (soft delete) |             |

### Особенности:

- Все поля уникальны для предотвращения дублирования поставщиков
- Soft delete для сохранения истории
- Комментарий позволяет добавлять дополнительную информацию

## shelves - Стеллажи склада

**Файл миграции:** `2025_09_22_105631_create_shelves_table.php`

### Структура таблицы:

| Колонка    | Тип         | Описание                | Индекс/Ключ |
|------------|-------------|-------------------------|-------------|
| id         | bigint      | ID стеллажа             | PRIMARY     |
| title      | string(255) | Название/номер стеллажа |             |
| created_at | timestamp   | Дата создания           |             |
| updated_at | timestamp   | Дата обновления         |             |

### Связи:

- Связь с `marketplace_order_items` через `shelf_id` (один ко многим)

### Особенности структуры:

1. **Поставщики** - справочник контрагентов поставляющих материалы
    - Хранят полную контактную информацию
    - Уникальность полей предотвращает дублирование
    - Комментарий для дополнительных заметок

2. **Стеллажи** - организация складского пространства
    - Простая структура для идентификации мест хранения
    - Используются для привязки товаров к конкретным местам на складе
    - Помогают в отслеживании местоположения материалов
