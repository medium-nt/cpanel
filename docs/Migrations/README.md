# Документация миграций базы данных

## Обзор структуры базы данных

База данных приложения предназначена для управления производственным процессом,
включая управление заказами с маркетплейсов, материалами, сотрудниками и
финансовыми транзакциями.

### Основные группы таблиц:

1. **Пользователи и управление доступом**
    - `users` - пользователи системы
    - `roles` - роли пользователей
    - `schedules` - графики работы сотрудников

2. **Маркетплейсы и заказы**
    - `marketplace_orders` - заказы с маркетплейсов
    - `marketplace_order_items` - элементы заказов
    - `marketplace_items` - товары маркетплейсов
    - `marketplace_order_history` - история изменений заказов
    - `marketplace_supplies` - поставки на маркетплейсы

3. **Материалы и склад**
    - `materials` - материалы
    - `type_materials` - типы материалов
    - `movement_materials` - движения материалов
    - `material_consumptions` - расход материалов
    - `shelves` - стеллажи склада
    - `suppliers` - поставщики
    - `sku` - SKU товаров
    - `stacks` - стопки материалов

4. **Финансы и мотивация**
    - `transactions` - финансовые транзакции
    - `rates` - ставки оплаты
    - `motivations` - мотивационные бонусы

5. **Производство**
    - `orders` - производственные заказы
    - `inventory_checks` - инвентаризации
    - `inventory_check_items` - элементы инвентаризации

6. **Справочники**
    - `settings` - настройки системы
    - `cache` - кэш системы
    - `jobs` - очереди задач
