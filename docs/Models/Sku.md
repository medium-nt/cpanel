# Sku

## Назначение модели

Модель `Sku` представляет артикулы (SKU) товаров для разных маркетплейсов.
Связана с таблицей `skus` в базе данных.

## Атрибуты

### Fillable (массово присваиваемые)

- `item_id` - ID товара (MarketplaceItem)
- `sku` - артикул товара на маркетплейсе
- `marketplace_id` - ID маркетплейса

### Отсутствуют

- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

### BelongsTo (многие к одному)

- `item()` - товар, к которому относится артикул

## Скоупы и методы

Отсутствуют специфические скоупы или методы.

## Особенности реализации

- Позволяет одному товару иметь разные артикулы на разных маркетплейсах
- Структура поддерживает:
    - Множественные SKU для одного товара
    - Различные артикулы для разных площадок
    - Гибкую систему идентификации товаров
- `marketplace_id` позволяет:
    - Разделять артикулы по площадкам
    - Избегать конфликтов SKU
    - Организовывать мультиплатформенную продажу
- Связь с товаром позволяет:
    - Получать полную информацию о продукте
    - Агрегировать данные о продажах
    - Управлять остатками по товару независимо от маркетплейса
