# Supplier

## Назначение модели

Модель `Supplier` представляет поставщиков материалов и сырья для производства.
Связана с таблицей `suppliers` в базе данных.

## Атрибуты

### Fillable (массово присваиваемые)

- `title` - название поставщика
- `phone` - контактный телефон
- `address` - адрес поставщика
- `comment` - дополнительная информация или заметки

### Traits

- `SoftDeletes` - мягкое удаление (поставщики не удаляются физически из БД)

### Отсутствуют

- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

### HasMany (один ко многим)

- `orders()` - заказы на поставку материалов от этого поставщика

## Скоупы и методы

Отсутствуют специфические скоупы или методы.

## Особенности реализации

- Использует мягкое удаление через трейт SoftDeletes
- Хранит контактную информацию для связи с поставщиками
- Поле `comment` позволяет хранить дополнительную информацию:
    - Условия поставки
    - Особенности работы
    - Примечания о качестве материалов
- Связь с заказами позволяет отслеживать:
    - Историю заказов у поставщика
    - Объемы поставок
    - Частоту заказов
- При мягком удалении сохраняет всю историю заказов поставщика
