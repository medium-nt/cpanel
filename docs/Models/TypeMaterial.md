# Модель TypeMaterial

**Путь:** `app/Models/TypeMaterial.php`

**Назначение:** Модель для классификации типов материалов в системе

**Таблица в БД:** `type_materials`

## Атрибуты

### Fillable (массово назначаемые)

- `title` - Наименование типа материала

## Особенности использования

1. **Классификатор:** Используется для группировки материалов по типам
2. **Простая структура:** Содержит только поле с названием типа
3. **Связь с материалами:** Через модель Material осуществляется связь HasMany

## Возможные типы материалов

На основе названия модели можно предположить следующие типы:

- Ткани
- Нитки
- Фурнитура
- Уплотнители
- Крепеж
- И другие категории материалов

## Примеры использования

```php
// Получение всех типов материалов
$types = TypeMaterial::all();

// Создание нового типа материала
$newType = TypeMaterial::create(['title' => 'Ткани']);

// Получение материалов определенного типа
$type = TypeMaterial::find($typeId);
$materials = $type->materials; // Предполагается, что в модели Material есть связь type()

// Поиск типа по названию
$fabricType = TypeMaterial::where('title', 'Ткани')->first();
```
