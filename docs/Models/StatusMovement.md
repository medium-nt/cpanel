# StatusMovement

## Назначение модели

Модель `StatusMovement` определяет возможные статусы заказов и их отображение в
интерфейсе. Это статическая модель-справочник, не связанная напрямую с таблицей
в БД.

## Особенности реализации

- Модель-справочник, содержащая только константы
- Используется другими моделями (особенно Order) для определения названий и
  цветов статусов
- Статусы охватывают полный жизненный цикл заказа от создания до завершения или
  утери
- Цвета бейджей соответствуют Bootstrap классам для удобного отображения в UI
- Не является полноценной моделью Eloquent с таблицей в БД
