# Модель Role

**Путь:** `app/Models/Role.php`

**Назначение:** Модель для хранения ролей пользователей в системе

**Таблица в БД:** `roles`

## Атрибуты

### Fillable (массово назначаемые)

- `name` - Название роли

## Связи (Relationships)

### HasMany (имеет много)

- `users()` - Пользователи с данной ролью (связь с моделью User)

## Особенности использования

1. **Простая модель:** Содержит только базовые поля для определения ролей
2. **Ролевая система:** Используется для определения прав доступа пользователей
   в приложении
3. **Типы ролей:** В коде упоминаются следующие типы ролей:
    - `admin` - Администратор
    - `cutter` - Резчик
    - `seamstress` - Швея
    - `storekeeper` - Кладовщик
    - `otk` - Сотрудник отдела технического контроля

## Примеры использования

```php
// Получение всех ролей
$roles = Role::all();

// Получение пользователей определенной роли

// Создание новой роли
$role = Role::create(['name' => 'manager']);
```
