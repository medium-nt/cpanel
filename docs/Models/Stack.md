# Stack

## Назначение модели

Модель `Stack` представляет стек или пачку заказов, назначенных швее для
выполнения. Связана с таблицей `stacks` в базе данных.

## Атрибуты

### Fillable (массово присваиваемые)

- `seamstress_id` - ID швеи (User)
- `stack` - текущее количество заказов в стеке
- `max` - максимальное количество заказов, которое может быть в стеке

### Отсутствуют

- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

Отсутствуют явные связи, но неявно связана с моделью `User` через поле
`seamstress_id`.

## Скоупы и методы

Отсутствуют специфические скоупы или методы.

## Особенности реализации

- Простая модель для отслеживания загрузки швей
- Позволяет контролировать количество заказов, назначенных каждой швее
- Поле `max` определяет максимальную загрузку для швеи
- Может использоваться для автоматического распределения заказов между швеями
- Нет явных связей с другими моделями (только через seamstress_id)
