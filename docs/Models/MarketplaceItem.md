# MarketplaceItem

## Назначение модели

Модель `MarketplaceItem` представляет товары, которые продаются на
маркетплейсах. Связана с таблицей `marketplace_items` в базе данных.

## Атрибуты

### Fillable (массово присваиваемые)

- `title` - название товара
- `width` - ширина товара
- `height` - высота товара

### Отсутствуют

- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

### HasMany (один ко многим)

- `marketplaceOrderItem()` - элементы заказов с маркетплейсов, содержащие этот
  товар
- `sku()` - SKU артикулы этого товара для разных маркетплейсов
- `consumption()` - норма расхода материалов для производства этого товара

## Скоупы и методы

Отсутствуют специфические скоупы или методы.

## Особенности реализации

- Простая модель для хранения информации о товарах
- Используется для расчета расхода материалов при производстве
- Связана с заказами через элемент заказов (MarketplaceOrderItem)
- Может иметь несколько SKU для разных маркетплейсов
