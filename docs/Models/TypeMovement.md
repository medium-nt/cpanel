# TypeMovement

## Назначение модели

Модель `TypeMovement` определяет типы перемещения материалов на складе и в
производстве. Это статическая модель-справочник, связанная с таблицей
`type_movements` в базе данных.

## Константы

### TYPES (типы перемещений)

```php
const TYPES = [
    1 => 'Поступление от поставщика',
    2 => 'Отгрузка на производство',
    3 => 'Списание по заказу',
    4 => 'Брак на производстве',
    5 => 'Возврат брака поставщику',
    6 => 'Списание недосдачи с цеха',
    7 => 'Остатки на производстве',
    8 => 'Утилизированные остатки',
];
```

## Атрибуты

### Свойства (Properties)

- `status_id` (int) - ID типа перемещения

### Отсутствуют

- `fillable` - отсутствуют массово присваиваемые поля
- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

Отсутствуют явные связи.

## Скоупы и методы

### Accessors (мутаторы/доступы)

- `getTypeNameAttribute(): string` - возвращает название типа перемещения
    - Использует константу TYPES для получения названия по status_id
    - Если статус не найден, возвращает пустую строку

## Особенности реализации

- Модель-справочник для классификации движения материалов
- Охватывает все основные этапы жизненного цикла материалов:
    - Поступление от поставщиков
    - Передача в производство
    - Списание (в том числе брак)
    - Возвраты
    - Утилизация
- Используется моделью Order через атрибут `type_movement_name`
- Предоставляет централизованное хранилище названий типов перемещений
- Может быть расширена для добавления новых типов движения материалов
