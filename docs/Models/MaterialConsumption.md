# MaterialConsumption

## Назначение модели

Модель `MaterialConsumption` определяет норму расхода материалов для
производства товаров. Связана с таблицей `material_consumptions` в базе данных.

## Атрибуты

### Fillable (массово присваиваемые)

- `item_id` - ID товара (MarketplaceItem)
- `material_id` - ID материала (Material)
- `quantity` - количество материала, необходимое для производства

### Отсутствуют

- `hidden` - отсутствуют скрытые атрибуты
- `casts` - отсутствуют приведения типов

## Связи с другими моделями

### BelongsTo (многие к одному)

- `item()` - товар, для которого определяется норма расхода
- `material()` - материал, который используется для производства

## Скоупы и методы

Отсутствуют специфические скоупы или методы.

## Особенности реализации

- Промежуточная модель для связи товаров с материалами
- Определяет, сколько и каких материалов необходимо для производства каждого
  товара
- Используется для расчета стоимости производства и списания материалов при
  выполнении заказов
- Не использует трейт HasFactory
